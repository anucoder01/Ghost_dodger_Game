# 👻 Ghost Dodger

Ghost Dodger is a fast-paced, web-based survival game built with HTML5 Canvas, CSS, and JavaScript. Your goal is simple: survive as long as possible by dodging the falling ghosts! 

## 🎮 How to Play

- **Movement:** Use the **Left** and **Right** arrow keys to move your character.
- **Pause:** Press the **Escape (Esc)** key to pause or unpause the game.
- **Objective:** Avoid the falling ghosts (👾). The longer you survive, the higher your score!
- **Difficulty:** The game speed increases progressively every 5 seconds.

## 🌟 Features

- **Multiple Difficulties:** Choose between Easy, Medium, and Hard to set the initial ghost speed.
- **Character Skins:** Play as a Wizard (🧙‍♂️), Vampire (🧛‍♂️), or Zombie (🧟‍♂️).
- **Powerups:**
  - 🛡️ **Shield:** Grants temporary invincibility against ghost collisions for 5 seconds.
  - 🐢 **Slow:** Slows down the falling ghosts for 5 seconds to give you some breathing room.
- **High Score Tracking:** Your best survival time is saved locally in your browser so you can keep beating your record.
- **Pause Functionality:** Need a break? Just hit Esc to pause the action.

## 🚀 How to Run

1. Clone this repository or download the source code.
2. Simply double-click on the `Index.html` file to open it in any modern web browser.
3. No installation, server, or extra dependencies required!

## 🛠️ Technologies Used

- **HTML5:** Structure and Canvas element for rendering the game loop.
- **CSS3:** Styling, layout, animations, and gradients.
- **Vanilla JavaScript:** Game logic, rendering, event handling, and local storage integration.
